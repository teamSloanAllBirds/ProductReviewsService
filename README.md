## Requirements

- Node 12.16.3

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Seeding the Database
Please edit ./server/database/config.js before running
```sh
npm run seed
```

### Starting the server

From within the root directory:

Run the nodemon server:

```sh
npm run start
```

Run webpack:

```sh
npm run build
```
